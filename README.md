# shahrestanejajarm_bot

ربات تلگرام فارسی شهرستان جاجرم، بدون تبلیغ ManyBot.

## اجرای محلی
```bash
pip install -r requirements.txt
export BOT_TOKEN="TOKEN_HERE"
python bot.py
```

## اجرای Render
Build Command:
```bash
pip install -r requirements.txt
```

Start Command:
```bash
python bot.py
```

Environment Variable:
```text
BOT_TOKEN = توکن ربات از BotFather
```
